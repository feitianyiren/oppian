Follow this link to download it from iTunes.

Oppian created the initial end-to-end system architecture for Live Talkback. In addition, we implemented the server back-end and the prototype of the iPhone client. To achieve this, we used Django and the iPhone SDK along with Amazon EC2 and RightScale services.

We're proud to have helped Live Talkback win 3rd place in the O2 Litmus app showdown.
